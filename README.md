# S-ZombiesAsteroids
Created by Sam Alonso 2020

### Installation

Download [Unity](https://unity3d.com/get-unity/download/) v 2019.4.11f1 LTS

### Configure

Set resolution to Iphone X/XS portrait mode

### How to play

```sh
WASD -> Movement
SPACE -> Shoot
```
